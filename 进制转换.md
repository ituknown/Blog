# 前言

在我们的各种编程语言中，整数一般有四种表示方式。分别是二进制、八进制、十进制以及十六进制。

所谓的几进制就是逢几进一的意思。比如我们在日常生活中的数学表示法就是逢十进一，也就是所谓的十进制。

对于各种进制的运算规则是：

1）二进制：只有0和1，逢2进一。
2）八进制：0-7，满8进1。在编程中以数字 0 开头表示。
3）十进制：0-9，满10进1.
4）十六进制：0-9及A-F满，16进1。以 0x 或 0X 开头表示（A表示10，一次类推，F表示 15）
